## 模板结构

每个示例包含以下内容：

```
example_主题名称/
├── prompt.md          # 完整的提示词文档
└── images/            # 生成的图片示例（可选）
    ├── basic.png      # 基础版效果
    ├── enhanced.png   # 增强版效果
    └── pro.png        # 专业版效果
```

## 模板规范要点

### 1. 元信息
- 创建日期
- 标签（便于搜索和分类）

### 2. 需求摘要（8 大维度）
- 主体、图片类型、视觉风格
- 场景环境、光线氛围、视角构图
- 色彩倾向、宽高比、目标平台

### 3. 多版本提示词（中英文双语）
- **基础版**：15-25词，快速测试
- **增强版**：30-50词，日常使用
- **专业版**：50-80词，高质量输出

> 📝 每个版本都提供中英文两种语言的提示词

### 4. 平台适配
- 针对不同平台的格式优化
- 说明各平台的特点

### 5. 辅助信息
- 使用说明
- 迭代记录
- 优化建议

---

## 如何使用

1. 浏览示例，了解提示词的结构和风格
2. 复制感兴趣的提示词到 AI 图像生成平台测试
3. 参考格式创建自己的项目（保存到 `projects/` 目录）
