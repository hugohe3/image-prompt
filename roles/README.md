# 🎭 角色系统

> Image Prompt Master 采用多角色协作架构，将复杂的提示词生成任务分解为清晰的职责边界。

## 输出说明

本系统生成的是 **Markdown 格式的优化提示词**，用户可：
- 📋 复制到 Gemini / ChatGPT / Midjourney 等 Web 平台使用
- 📁 保存到 `projects/` 目录管理
- 🔌 如有 MCP 工具，可直接调用生成图片

---

## 角色概览

| 角色 | 职责 | 触发时机 |
|------|------|----------|
| **Strategist** | 需求分析与确认 | 用户首次描述图像需求时 |
| **Prompt_Architect** | 生成多版本优化提示词 | 需求确认完成后 |

---

## 工作流程

```
用户描述初步想法
       │
       ▼
┌─────────────────────────────────┐
│  Strategist (策略师)             │
│  - 复述用户意图                  │
│  - 评估主题难度                  │
│  - 提出 3-5 个确认问题           │
│  - 提供选项和专业建议            │
└─────────────────────────────────┘
       │
       ▼ 用户确认完成
       │
┌─────────────────────────────────┐
│  Prompt_Architect (提示词架构师) │
│  - 生成需求摘要表                │
│  - 输出基础版/增强版/专业版      │
│  - 提供平台适配版本              │
│  - 支持迭代优化                  │
└─────────────────────────────────┘
       │
       ▼
  多版本优化 Prompt
```

---

## 核心原则

### 遵循公式
```
完美图像 = 好的主题 × 精心设计的提示
```

### 交互原则
1. **每次只问 3-5 个最关键的问题**，避免问题过多
2. **根据用户描述的详细程度调整提问量**：描述越详细，问题越少
3. **始终提供推荐选项**，降低用户决策负担
4. **快速响应**：用户选择后立即输出提示词
5. **支持快捷回复**：用户可直接说"都听你的"让你全权决定
